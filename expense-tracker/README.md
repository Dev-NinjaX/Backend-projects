Expense Tracker (Based on CLI)

This is a basic command-line expense tracker built in Java.  
I made this project to practice core Java concepts and understand how to structure a small application.

# What it does

- Add expenses
- View all saved expenses
- update expenses
- Show a monthly summary for expenses
- Delete expenses
- Save data to a text file so it doesn't disappear after closing the program

# Why I built this

I wanted to get comfortable with:
- Using ArrayList
- Handling user input with Scanner
- Reading and writing files
- Organizing code into classes

# How to run

1. Clone the repository
2. Go into the project folder
3. Compile:
   javac ExpenseTracker.java
4 Run:
   java ExpenseTracker
